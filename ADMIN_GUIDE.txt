SHORT FORMAT
------------
Format: roll|c:cgpa|g1:gpa1,g2:gpa2,...|s:subjects

Examples:
731559|c:3.25|g1:3.00,g2:r,g3:2.85|s:25912,26731
731625|g1:3.06,g2:3.30,g3:2.76,g4:3.25
500104|c:3.52|g1:3.24,g2:3.05,g3:2.67,g4:3.29

Legend:
c = CGPA
g1-g8 = GPA for semesters 1-8
r = Referred
n = Not Published Yet
s = Referred Subjects

TROUBLESHOOTING
---------------

Problem: Upload fails
Solution:
- Check database status (green dot in hidden.html)
- Wait 5 seconds and retry
- System auto-retries 3 times
- Check terminal logs for details

Problem: Database not ready
Solution:
- Wait 10 seconds for connection
- Refresh page
- Check hidden.html for status

Problem: Invalid format
Solution:
- Use format.html to convert
- Verify 6-digit roll number
- Check for missing pipes (|)
- Remove extra spaces
