1. MAINTENANCE MODE SYSTEM
========================================

ADMIN CONTROL PANEL
-------------------
Location: admin.html
Feature: Toggle switch for maintenance mode

HOW TO USE:
1. Login to admin panel (admin-login.html)
2. Find "Maintenance Mode" toggle at the top
3. Click toggle to switch ON/OFF
4. Status shows: ON (Red) or OFF (Green)

WHEN TO USE:
- Uploading new results
- Database maintenance
- System updates
- Emergency downtime

WHAT HAPPENS WHEN ON:
- All visitors redirected to maintenance.html
- Shows friendly maintenance page
- Displays "uploading new results" message
- Students know to check back soon

WHAT HAPPENS WHEN OFF:
- Normal website operation
- All pages accessible
- Result checking works normally

TECHNICAL DETAILS:
- Uses localStorage to store state
- Checked on page load
- Works across all pages
- Admin can toggle anytime

SCENARIO: Uploading New Results
--------------------------------

STEP 2: UPLOAD RESULTS

- Upload via admin panel
- Monitor progress

- Toggle maintenance OFF
- Website returns to normal
- Students can check results
- System fully operational

- Small upload (100 records): 5 minutes

========================================
TROUBLESHOOTING
========================================

PROBLEM: Maintenance mode won't turn off
SOLUTION: Clear localStorage, refresh page, toggle again

PROBLEM: Students still see old results
SOLUTION: Clear browser cache, hard refresh (Ctrl+F5)

PROBLEM: SEO not improving
SOLUTION: Submit sitemap to Google Search Console, wait 1-2 weeks

PROBLEM: Maintenance page not showing
SOLUTION: Check localStorage, verify maintenance.html exists
